# Modern Portfolio Website


## Features

- **Responsive Design**: Fully mobile-friendly layout
- **Modern UI**: Clean design with gradient accents and subtle animations
- **Dark Mode**: Toggle between light and dark themes
- **Smooth Animations**: Scroll animations and transitions
- **Contact Form**: Interactive form with validation
- **Project Showcase**: Responsive grid layout for projects

## Sections

1. **Hero Section**: Introduction with name, title, and tagline
2. **About Me**: Background information, skills, and profile photo
3. **Projects**: Showcase of work with descriptions and links
4. **Contact**: Form and contact information
5. **Footer**: Social media links and copyright information

## Technologies Used

- HTML5
- CSS3 (with CSS variables for theming)
- JavaScript (Vanilla JS, no frameworks)
- Font Awesome for icons
- Google Fonts (Poppins)

## Customization

To customize this template for your own use:

1. Replace placeholder images with your own photos
2. Update personal information in the HTML file
3. Modify the color scheme by changing CSS variables in the `:root` selector
4. Add your own projects to the projects section
5. Update social media links in the footer

## Browser Support

This template works in all modern browsers including:
- Chrome
- Firefox
- Safari
- Edge
